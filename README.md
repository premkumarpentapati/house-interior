# House Interior — Modern Warm Minimalist Duplex

A complete architecture-grade interior design proposal for a duplex residence, published as a static website for **GitHub Pages**.

## Live site

Enable GitHub Pages from the `main` branch root, then open:

`https://premkumarpentapati.github.io/house-interior/`

## What’s included

- Design philosophy (Modern Warm Minimalist)
- Existing-house analysis method + placeholders for your photos
- Working floor plans (SVG) with **all beds locked South → North**
- **Five original concepts** within the same theme
- Room-by-room designs with visualizations
- Asian Paints specifications (Royale + Royale Play textures)
- Material boards, lighting plan, furniture plan
- Cost optimization comparisons
- Before & after framework
- Gallery + final recommendation (**Concept 02 — Oat & Oak**)

## Important note on source files

At build time the repository did **not** yet contain:

- First / second floor plan PDFs
- Current interior / exterior photographs
- Sample website screenshots

Designs therefore use a carefully documented **working duplex layout** typical of contemporary Indian homes. When you upload:

1. Place PDFs in `assets/plans/`
2. Place existing photos in `assets/images/before-after/` and `assets/images/existing/`
3. The concept language, paint system, and lighting strategy remain valid — dimensions get recalibrated
